A work in progress website that will soon track all relevant data about the cryptocurrency "Kadena" (KDA). With this side project I wanted to use some of my extensive cryptocurrency and NFT knowledge and challenge myself to build a useful application that can be used to track this emerging ecosystem. This application aims to track not only all tokens that are currently actively traded on the Kadena blockchain but will also scrape for relevant information on projects that are set to launch on the chain. HRM

A JAM stack blog website created for a University assignment complete with a Sanity "headless" CMS backend and Next.js front end using incremental static regeneration to enhance speed of page loads and updates. Fast updates can be made to blog posts from the in-built Sanity Studio which can be accessed only by authorised parties upon typing /studio at the end of the website's URL. This means that the backend runs on the same hosting platform as the blog website allowing significant cost saving.
