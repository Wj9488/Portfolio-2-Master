ABOUT THE PROJECT

A JAM stack blog website created for a University assignment complete with a Sanity "headless" CMS backend and Next.js front end using incremental static regeneration to enhance speed of page loads and updates. Fast updates can be made to blog posts from the in-built Sanity Studio which can be accessed only by authorised parties upon typing /studio at the end of the website's URL. This means that the backend runs on the same hosting platform as the blog website allowing for non-technical team members to update content easily.
